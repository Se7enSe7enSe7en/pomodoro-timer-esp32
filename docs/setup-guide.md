# Setup Guide

## Install these extensions
- esp-idf
- cmake tools

## Configure ESP-IDF extension
1. press F1 (ctrl + P)
2. `>configure esp`, press on "ESP-IDF: Configure ESP-IDF Extension"

## ESP-IDF prerequisites for linux/macos

- https://docs.espressif.com/projects/esp-idf/en/latest/esp32/get-started/linux-macos-setup.html#get-started-prerequisites
